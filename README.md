## ğŸ’¡ How to Run Examples

### Method 1: Browser Console
1. Open your browser (Chrome, Firefox, Edge)
2. Press `F12` to open Developer Tools
3. Go to the **Console** tab
4. Copy and paste JavaScript code
5. Press `Enter`

### Method 2: HTML File
1. Create or open an HTML file from `workarea/`
2. Look for code between `<script>` tags
3. Double-click the file to open in your browser
4. Check the console for output
