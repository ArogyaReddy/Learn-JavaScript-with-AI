Functions: Declaring and Calling - Function Types

Explanation

Functions are reusable code blocks performing tasks. Declaring creates the function; calling executes it. JavaScript has three types: function declarations (hoisted, traditional), function expressions (stored in variables), and arrow functions (modern, concise).

Example 1: Function Declaration (Hoisted)

Can be called before it's defined in code because it's "hoisted" to the top.
