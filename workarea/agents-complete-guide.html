Understanding Custom GitHub Copilot Agents

What is AGENTS.md?

AGENTS.md is a file that defines custom agent behaviors for GitHub Copilot. Unlike SKILLS (which are called on-demand), AGENTS are always active and modify how Copilot behaves in your workspace.

üéØ SKILLS

  • Location: .github/skills/
  • Format: Folder with SKILL.md
  • Activation: On-demand (user triggers)
  • Purpose: Specialized tasks
  • Example: "tip of the day"

ü§ñ AGENTS

  • Location: Root or .github/
  • Format: Single AGENTS.md file
  • Activation: Always active
  • Purpose: Modify Copilot behavior
  • Example: Auto mentoring mode
